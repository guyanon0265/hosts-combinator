# hosts-combinator
Repo to hold combined hosts list to block explicit sites.  
This repo makes automated daily checks for updates in the source lists.  
Please contribute any URLs that need to be added through the links below, not here.

## combined hosts file
To use the combined hosts file for your own purposes, look in lists/hosts  
There is also a sqlite3 database containing all the raw URLs available in sqlite/blockedurls.sqlite3
